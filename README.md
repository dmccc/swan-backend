## Quick Start

Click the `Use this template` button to start using the **swan-backend**!

It's that easy!

## Module

You should start `swan-config-app` first, then start `swan-eureka-app`, and finally start other services.

```
swan-backend
├── swan-appendix-app
├── swan-auth-app
├── swan-config-app
├── swan-eureka-app
├── swan-gateway
├── swan-stater-auth
```
